# 数据同步任务指令

## 项目背景
需要从DeBank网站抓取协议元数据，并每天定时同步到本地数据库。

## DeBank网站分析
- 网址: https://debank.com
- 协议列表页: https://debank.com/protocols
- 需要抓取的信息:
  1. 协议名称
  2. 协议类型（DEX、Lending、Yield等）
  3. 支持的链（ETH、BSC、Polygon等）
  4. 总锁定价值（TVL）
  5. 协议官网
  6. 协议描述
  7. 支持的代币

## 技术实现要求

### 1. 网页抓取器
- 使用Colly或GoQuery库
- 处理JavaScript渲染（可能需要headless browser）
- 处理分页加载
- 处理反爬虫机制（User-Agent、延迟）

### 2. 数据清洗
- HTML标签清理
- 数据格式标准化
- 空值处理
- 数据验证

### 3. 定时任务
- 每天凌晨2点执行
- 支持手动触发
- 任务状态跟踪
- 错误重试机制

### 4. 数据存储
- 协议基本信息表
- 协议链关联表
- 协议代币关联表
- 同步历史记录表

### 5. 错误处理
- 网络错误重试（最多3次）
- 数据解析错误记录日志
- 数据库错误回滚
- 任务超时终止

### 6. 性能优化
- 并发抓取（控制并发数）
- 数据批量插入
- 内存使用优化
- 抓取速度控制

## 输出要求
创建完整的、可运行的协议元数据同步系统。
